Assignment.

Functional requirements

Our client application requires data from different sources. A REST API service needs to be created to gather data from these sources. This will decrease the amount of calls on the client application.

Retrieving data from different sources can be time consuming. Implement a solution when retrieving data from different sources is done parallel.

By introducing this new REST API service we would also like to monitor this service.

Technical requirements

  • Introduce an endpoint to retrieve data which consist of data from different sources (you may define the url yourself)
  • Return a JSON response to clients visiting the endpoint, you may define a model for the result yourself.
  • The endpoints of the sources must be configured in a properties file
  • Introduce a new endpoint that exposes a JSON view of application version, number of consumer invocations and average request processing time.
  • Add documentation for the usage of the endpoints

Bonus assignment

Front end code is not the main priority of this assignment, we will be focusing on your backend implementation.

  • Bonus assignment 01: Create a simple Single Page Application (SPA) which connects to your endpoints and outputs the data
  • Bonus assignment 02: Add additional client side features

Conclusion

By no means is the goal to get a solution covering all special cases in a 100% robust way. Requiring this would be naive. At least the functions described shall be error free when used correctly, how everything else is handled is subject to creativity and ambition.

We will pay attention to code structure, threading and understanding of the problem, as well as the discussion around design and choice of data structures.

Please note: This type of exhaustive specification is extremely rare at klm.com and does by no means represent the way we do or want to work. In reality the “specification” is almost always verbal and implicit and it is up to the individual developers or groups to formulate code that best represents the solution.
